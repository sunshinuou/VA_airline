1. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

2. **Data**
   - The dataset we use is `dataset/data2.csv`, it has been set in project.py

3. **Run the code**
   ```bash
   python project.py
   ```

4. **Access the dashboard**
   - Open the browser and navigate to: `http://127.0.0.1:8050/`
